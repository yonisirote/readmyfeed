# ReadMyFeed Expo Client

## Quick start

```bash
npm install
npm run start
```

Open the Expo app, paste your API key, and fetch the first page of tweets.

## CLI fetch (no device)

```bash
API_KEY=your_api_key_here npm run fetch:cli
```

Or pass the key directly:

```bash
npm run fetch:cli -- --api-key "your_api_key_here"
```
